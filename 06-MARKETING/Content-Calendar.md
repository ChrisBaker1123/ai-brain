## Weekly Template

| Day | Channel | Content Type | Theme |
|-----|---------|-------------|-------|
| Monday | LinkedIn | Educational post | AI tip or how-to |
| Tuesday | LinkedIn | Template preview | Show a real template |
| Wednesday | Blog (when live) | Long-form article | Deep dive on topic |
| Thursday | LinkedIn | Compliance/trust | Address advisor fears |
| Friday | LinkedIn | Personal/story | Behind the scenes |

## Monthly Priorities

### Month 1: Foundation
- [ ] Set up blog
- [ ] Write 4 foundational blog posts
- [ ] Post daily on LinkedIn for 30 days
- [ ] Create og:image for social sharing
- [ ] Set up email sequences (Resend)

### Month 2: Growth
- [ ] Guest post on advisor-focused publications
- [ ] Create 2 additional comparison pages
- [ ] Launch email nurture sequence
- [ ] Start advisor community engagement
- [ ] Create video walkthrough

### Month 3: Scale
- [ ] Analyze what content performs best
- [ ] Double down on top-performing formats
- [ ] Create firm-type specific landing pages
- [ ] Develop case study (if users available)
- [ ] Plan conference presence
