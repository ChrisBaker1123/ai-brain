## The Market
The AI-for-financial-advisors space is emerging. Most advisors use generic AI tools (ChatGPT, Copilot) without structure, compliance awareness, or personalization.

## Direct Competitors

### 1. Generic AI Tools (ChatGPT, Claude, Copilot)
- **Strengths**: Free/cheap, powerful models, broad capabilities
- **Weaknesses**: No advisor-specific templates, no compliance references, no personalization, requires prompt engineering skill
- **Our differentiation**: We work WITH these tools, not against them. We provide the structure; they provide the intelligence.

### 2. FP Alpha
- **Focus**: Financial planning analysis automation
- **Approach**: Automated document analysis (tax returns, estate docs)
- **Weaknesses**: Different use case (analysis vs. content creation), expensive, steep learning curve
- **Our differentiation**: Copy-paste simplicity, broader use cases, compliance-aware

### 3. VRGL (formerly Jump)
- **Focus**: Portfolio analysis and proposal generation
- **Approach**: Upload competitor statements, generate proposals
- **Weaknesses**: Narrow use case (prospecting only), requires data upload
- **Our differentiation**: No data upload needed, broader template categories

### 4. Advisor Prompt Libraries (Various)
- **Focus**: Free or paid prompt collections (PDFs, spreadsheets)
- **Approach**: Static lists of prompts without personalization
- **Weaknesses**: No auto-injection of profile data, no compliance references, no updates, no UI
- **Our differentiation**: Dynamic personalization, compliance guardrails, searchable UI, favorites, history

### 5. Riskalyze / Nitrogen AI Features
- **Focus**: Risk assessment + AI-assisted communications
- **Approach**: Built into existing platform
- **Weaknesses**: Locked into their ecosystem, limited template variety
- **Our differentiation**: Platform-agnostic, works with any AI tool, broader categories

## Our Positioning (updated 2026-03-14)
**"The AI layer for everything your existing systems don't cover."**

Key differentiators:
1. **Gap-filler, not replacement** — We complement eMoney, Holistiplan, Riskalyze, Orion — not compete with them
2. **Workflow-organized** — Templates organized by what advisors DO (meeting prep, follow-ups, emails) not by planning function
3. **Copy-paste model** — Client data never enters our platform
4. **Compliance-aware** — SEC/FINRA/Reg BI references in every template
5. **Zero learning curve** — No prompt engineering required
6. **Works WITH existing AI tools** — Copilot, ChatGPT, or any AI assistant

The "stack" positioning:
```
Planning:     eMoney/RightCapital/MoneyGuide    → builds the plan
Tax:          Holistiplan/FP Alpha              → analyzes the returns
Risk:         Nitrogen/Riskalyze                → assesses the risk
Meetings:     Jump/Zocks/Pulse360               → documents the meetings
Proposals:    VRGL/Kwanti                       → generates the proposals
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
MISSING:      ★ Advisor Intelligence ★          → creates the content
```

## Market Opportunity
- ~300,000 financial advisors in the US, 85% view AI as a "help" (Advisor360 2025)
- 43% already use AI for email drafting (Horsesmouth 2025) — but with generic tools
- $363M+ in recent VC funding for advisor AI tools validates the market
- No funded competitor fills the communication/content gap — all focus on analysis, meetings, or planning
- 15-25 hours/week of advisor time is spent on communication/documentation that no system covers
