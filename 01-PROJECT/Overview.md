# Advisor Intelligence — Overview

#project #overview

## What It Is
Advisor Intelligence is an AI-powered prompt toolkit SaaS for financial advisors. It provides **61 copy-paste templates** across 11 categories, a **Document Builder** with 20 templates, **18 How-To Guides**, **20 Action Plan scenarios**, and an **AI Coach** chatbot.

The core model is **copy-paste**: advisors select a template, it auto-injects their profile data, they copy it to their clipboard, and paste it into their firm's approved AI tool (Microsoft Copilot, ChatGPT, or Claude). Client data never enters the platform.
