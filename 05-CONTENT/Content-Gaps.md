2. **Template descriptions** — Some are generic. Could include "best for:" context.
3. **Document Builder** — Less polished than Templates section
4. **Action Plans** — Newest feature, may need more polish
5. **Quick Reference** — Very long (470 lines), could be overwhelming

## Social Proof Roadmap
When we get users:
1. First 5 users → Ask for quotes/testimonials
2. First firm adoption → Case study
3. Any notable advisor → "Used by [Name], CFP® at [Firm]"
4. Usage metrics → "Advisors have copied X templates this month"

**Rule**: Never fabricate. Only use real data from real users.
